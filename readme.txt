Cara Penggunaan Aplikasi:
1. Pastikan Anda memiliki koneksi internet, memiliki PHP laravel enviroment, terinstall gd library pada php Anda, dan memiliki SMTP server
2. Ekstrak source code pada direktori yang diinginkan
3. Buka terminal pada direktori tersebut
4. Ketikkan perintah "composer install" untuk mengunduh depedensi yang diperlukan
5. Salin file .env.example dengan nama .env
6. Atur file .env dengan menyesuaikan DB dan MAIL
8. Ketikan perintah "php artisan migrate --seed" untuk membuat database
9. Ketikan perintah "php artisan serve" untuk menjalankan Local Dev Server
10. Gunakan aplikasi dengan bijak
